# Операционна семантика

Среда - Environment - порменлива -> локация
Хранилище - Storage - локация -> стойност

```
E = [a : l_1, b : l_2] 
E' = E[l_3/a]
E`(a) = l_3  и  E`(c) = E(c), c!=a
```

```
S = [l_1 -> 5, l_2 -> 7]
S` = S[12/l_1]
S`[l_1] = 12,  S`(l) = S(l) l != l_1
```



X(a_1 = l_1,...)

- X е клас
- a_1 е атрибут
- l_1 е място в паметта


Нямат имена:
- Int(5)

`so, E, S ⊦ e: v, S'`
