### анонимна променлива

отбелязва се чрез "_"


### Равенство

- Формално: 
    - X = Y
    - X \= Y

- Аритметично:
    - X =:= Y
    - X =\= Y


### Оператор IS
```
Y is 4, X is 5+Y  
```
vs
```
X is 5+Y, Y is 4  // error: Y not defined
```

Трябва отдясно да го знаеш за `somethig is known/5/X`, където X се знае
